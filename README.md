## Commands ðŸ—£

* `"open tcgenius"` - Open the skill on your Alexa

* `"help"` - Get help

* `"Find me a {card} card"` - Find up to 5 random cards of a given type
   * `energy` - An energy card
	* `trainer` - A trainer card
	* `pokemon` - Pokemon card
